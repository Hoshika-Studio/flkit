# Roadmap

FLKit is starting as a focused Flutter project generator. The first goal is to
make new repositories quick to create, predictable to maintain, and easy for
humans or AI agents to understand.

## v0.2

- Add `flkit add feature <name>`.
- Generate feature folders with presentation, application, domain, and i18n
  files.
- Add non-interactive flags for feature generation.
- Improve generated README content inside starter projects.

## Future Ideas

- Add more starter templates.
- Add more language presets.
- Add JSON output for automation and AI workflows.
- Add project health checks for generated Flutter apps.
- Add update helpers for existing FLKit-generated projects.
